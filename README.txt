This program is designed to accept a CSV and convert that CSV into SQL code.
The CSV must be formatted in a particular way in order for the application to work correctly. 


Example:

ProductName,SKU,Quantity,DateAdded,Color,Size
Shirt,SKU00001,10,2022-03-15,Green,L
Shirt,SKU00002,5,2022-03-16,Red,M
Pants,SKU00003,2,2022-03-17,Green,M
Hat,SKU00004,8,2022-03-18,Black,XL
Pants,SKU00005,12,2022-03-19,Blue,S


You can add as many lines as you want as long as they follow this format
